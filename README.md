# Niyamit - Your Personal Life Organizer

Niyamit (Sanskrit for "Disciplined/Regular") is a modern, premium web application designed to help you organize your daily schedule, habits, and tasks with beautiful analytics.

## Features

- **Dashboard & Analytics**: 
  - Visual breakdown of your consistency (last 30 days).
  - Current streak tracking for habits.
  - "Attention Needed" section for frequently missed tasks.
  - Focus area distribution (Work vs Health vs Home).

- **Calendar & Schedule**:
  - Beautiful monthly calendar view.
  - Interactive daily task list.
  - Quick "Add Task" functionality.
  - Task categorization (Health, Work, Study, Home, Social) with color coding.

## Tech Stack

- **Framework**: React + Vite
- **Styling**: Vanilla CSS (Premium Dark Theme with Glassmorphism)
- **Icons**: Lucide React
- **Charts**: Recharts
- **Date Handling**: date-fns

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:5173](http://localhost:5173) in your browser.

## Customization

- Modify `src/index.css` to change the color palette.
- Edit `src/data/mockData.js` to change the initial data generation logic.
